# Stock Market Time-Series Analysis using Prophet

Stock market time-series analysis using Python and Prophet. Applies moving average filtering, ACF/CCF analysis, and one-step forecasting on NIFTY50 and major stocks to study trends, correlations, and short-term predictions with clear visual insights.

## Features
- Moving average filtering to smooth prices
- ACF / CCF analysis for temporal and inter-stock correlation
- One-step forecasting (Prophet) for prices and returns
- Visualizations: time series, ACF, CCF, forecast plots
